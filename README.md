## 🛠️ Skills Demonstrated

- **Cloud Security Configuration** — Azure NSGs, inbound/outbound rules, VM connectivity testing
- **Identity & Access Management** — User provisioning, SSPR, MFA, Conditional Access, PIM
- **Security Monitoring** — Sentinel SIEM setup, data connectors, analytics rules, workbooks
- **Compliance Management** — Sensitivity labels, insider risk policies, eDiscovery holds and search queries
- **PowerShell Administration** — Exchange Online module, audit log configuration via CLI
- **Microsoft Defender Suite** — Defender for Cloud, Defender for Cloud Apps, Defender Portal
- **Least Privilege Principles** — Role-based access control, just-in-time access via PIM
